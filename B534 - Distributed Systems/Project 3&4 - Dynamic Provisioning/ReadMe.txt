Note: Please start the Monitor before starting any of the Daemons.

Follow the below steps to execute the Monitor:

1. Untar "Monitor.tar" using the command "tar -xvf Monitor.tar" on the machine where you wish to run the Monitor.
2. Monitor directory will be created. Go to the newly created "Monitor" directory
3. Run the Shell Script "startMonitor.sh". This will start the Monitor process.


Follow the below steps to execute the Daemons:

1. Untar "Deamon.tar" using the command "tar -xvf Deamon.tar" on the machine where you wish to run the MPI program.
2. DeamonCode directory will be created. Go to the newly created "DeamonCode" directory
3. Run the Shell Script "startDeamon.sh". This will start the Daemon.
